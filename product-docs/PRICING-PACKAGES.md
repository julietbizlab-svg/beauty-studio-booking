# 報價方案

> **用途**：對外報價時快速說明三種方案。金額為建議區間，可依客戶規模與服務深度調整。  
> **原則**：客戶越不想自己碰技術，建置費越高；越需要長期代管，維護費越高。

---

## 方案一：Demo／樣品展示

### 適合誰

- 還在觀望、想先看實際畫面的潛在客戶
- 美業講座、說明會上的展示用
- 尚未決定簽約、只想體驗流程的店長

### 包含什麼

- 共用或示範版預約系統（非客戶專屬網域）
- 客人端：選服務、選日期、選時段、預約、取消
- 業主端：今日預約、服務管理、營業時段、店面設定
- 現場或線上 Demo 導覽（約 15～30 分鐘）

### 不包含什麼

- 客戶專屬 Notion／LINE／Cloudflare 設定
- 客戶品牌色、Logo、正式服務資料建置
- 上線部署與維護
- 客製功能開發

### 建議售價區間

**NT$ 0～3,000**（可免費，作為成交前展示）

### 後續維護費建議

不適用（未正式上線）。若 Demo 後簽約，改選下方方案二或三。

---

## 方案二：基礎買斷版

### 適合誰

- 一人工作室（美甲、美睫、霧眉、護膚等）
- 有基本 LINE 官方帳號，想開始線上預約
- 願意用業主端自己改服務與時段，或偶爾請你協助

### 包含什麼

- 客戶專屬系統上線（Notion + LINE LIFF + Cloudflare + GitHub Pages）
- 客人端、業主端完整功能
- 依客戶資料建置服務項目與營業時段
- 品牌色、公告、取消規則基本設定
- 上線驗收（客人端 + 業主端）
- 交付操作說明（非工程文件）
- 上線後 **30 天內** 小幅文字／服務調整（各 2 次以內）

### 不包含什麼

- 金流串接
- 多員工、多分店
- LINE 推播自動化
- 長期月維護（需另簽方案三）
- 大量客製開發

### 建議售價區間

**NT$ 18,000～38,000**（一次性建置費）

### 後續維護費建議

- 買斷後不強制維護
- 若客戶需要代改服務／時段／排查問題：**NT$ 1,500～3,000／次**，或改簽方案三

---

## 方案三：建置加維護版

### 適合誰

- 不想碰任何技術設定的店長
- 希望有人定期檢查系統、協助調整內容
- 重視穩定上線與長期有人可找

### 包含什麼

**建置（同方案二），另外包含：**

- 完整交付 SOP 執行與驗收紀錄
- 上線教學 1 次（約 60 分鐘，遠端或到府擇一）
- **每月**系統健康檢查（API、Notion 連線）
- **每月**小幅調整額度（例如：服務 3 項內、公告 2 次內、時段 1 次內）
- 優先問題排查（工作天回覆）

### 不包含什麼

- 金流、推播、多分店等大型功能
- 無限次大幅改版
- 24 小時 on-call
- 客戶帳號代付費用（Notion／Cloudflare 等第三方費用由客戶自行負擔）

### 建議售價區間

- **建置費**：NT$ 28,000～48,000（一次性）
- **或** 建置費略降 + 綁定維護約

### 後續維護費建議

**NT$ 2,000～6,000／月**（建議簽約至少 6 個月）

維護費通常包含：文字調整、服務項目更新、營業時間調整、基礎錯誤排查。  
超出額度或新功能開發，另報工時費。

## 三方案快速比較

| 項目 | Demo 展示 | 基礎買斷 | 建置＋維護 |
|------|-----------|----------|------------|
| 專屬上線 | ✗ | ✓ | ✓ |
| 客戶品牌設定 | ✗ | ✓ | ✓ |
| 上線驗收 | 展示用 | ✓ | ✓ |
| 操作教學 | 簡短導覽 | 說明文件 | 含 1 次教學 |
| 長期維護 | ✗ | 另計次 | 含月費 |
| 建議售價 | 0～3,000 | 18,000～38,000 | 28,000～48,000 + 月費 |

## 報價時記得說清楚的話

1. **第三方費用**：Notion、Cloudflare、GitHub 多為免費方案即可，但若客戶要進階方案，費用由客戶負擔。
2. **LINE 官方帳號**：需客戶自行申請或由你協助，帳號所有權建議歸客戶。
3. **資料所有權**：預約資料在客戶 Notion，屬於客戶。
4. **不含金流**：預約系統不代收費，到店付款或另議金流方案。
