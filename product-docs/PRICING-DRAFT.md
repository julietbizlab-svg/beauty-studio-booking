# 美業預約系統報價方案草稿

用途：整理這套系統未來販售時的報價方向，實際金額可依客戶需求調整。

## 一、方案 A：基礎建置版

適合：一人美甲、美睫、霧眉工作室。

內容：
- 客人端 LINE 預約頁
- 業主端 LINE 管理頁
- 服務項目設定
- 營業時段設定
- 基本品牌色設定
- GitHub Pages 前端部署
- Cloudflare Worker 後端部署
- Notion 資料庫設定

不包含：
- 金流
- 多員工
- 複雜報表
- 長期代管

建議售價：
NT$ 18,000～38,000

## 二、方案 B：標準品牌版

適合：想要更完整品牌形象的小型工作室。

包含基礎建置版，另外增加：
- 品牌色細緻調整
- 首頁公告設計
- 預約注意事項文案
- 客戶資料蒐集表
- 上線驗收清單
- 基礎教學 1 次

建議售價：
NT$ 38,000～68,000

## 三、方案 C：維護服務

適合：不想自己處理系統調整的業主。

月維護內容：
- 小幅文字修改
- 服務項目調整
- 營業時間調整
- 基礎錯誤排查
- 每月一次系統檢查

建議月費：
NT$ 2,000～6,000 / 月

## 四、不包含項目

以下需另行報價：
- 金流串接
- LINE 推播自動化
- 多員工排班
- 多分店管理
- 大型報表
- 客製化設計
- 客戶資料大量匯入
- 第三方平台整合

## 五、報價原則

- 客戶需要越少學習，建置費越高
- 客戶需要越多客製化，維護費越高
- 若客戶要自行修改，必須提供清楚教學文件
- 若客戶不想自行修改，必須簽月維護方案
